# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-12

### Added

- Initial `systems-thinking-dart` skill package.
- DART core procedure: Deconstruct, Analyze, Recognize, Test.
- Adapted Cynefin classification: Clear, Complicated, Complex, Chaotic.
- Output contract using a compact `<dart>` diagnostic block.
- Response templates for all four system types.
- Adapters for Claude, GPT, DeepSeek, Gemini, local small models, and reasoning-native models.
- Spanish examples for each system type.
- JSONL eval suite with 18 classification and activation cases.
- GitHub issue templates, PR template, lint workflow, and configurable eval workflow.
